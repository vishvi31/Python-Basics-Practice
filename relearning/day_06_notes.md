## Concepts Practiced

| Concept | What it does | Example |
|---|---|---|
| def | Define a function | def greet(): |
| parameter | Input to function | def greet(name): |
| return | Send value back | return a + b |
| default param | Value if not given | def greet(name, msg='Hi'): |
| multiple return | Return many values | return a, b, c |
| unpacking | Split return values | t, d, p = calculate(10, 2) |
| calling functions | Use function inside function | grade = get_grade(marks) |

---

## Why Functions?

| Without functions | With functions |
|---|---|
| Repeat same code everywhere | Write once, use anywhere |
| Hard to fix bugs | Fix in one place, fixed everywhere |
| Long messy code | Clean, readable, organised |
