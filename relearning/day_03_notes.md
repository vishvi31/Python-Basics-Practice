## Concepts Practiced

| Concept | What it does | Example |
|---|---|---|
| for loop | Repeats for each item | for i in range(5) |
| range(n) | Gives 0 to n-1 | range(5) = 0,1,2,3,4 |
| range(a,b) | Gives a to b-1 | range(1,6) = 1,2,3,4,5 |
| range(a,b,step) | Counts by step | range(0,20,2) = 0,2,4... |
| enumerate() | Gives index + value | for i, fruit in enumerate(list) |
| while loop | Repeats while True | while count <= 5 |
| break | Stops loop completely | if i == 5: break |
| continue | Skips current step | if i == 3: continue |
| nested loop | Loop inside loop | Multiplication table |

---

## For vs While - When to use which?

| Situation | Use |
|---|---|
| Know how many times to repeat | for loop |
| Repeat until something happens | while loop |
| Loop through a list | for loop |
| Wait for correct user input | while loop |
