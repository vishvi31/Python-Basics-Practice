## Concepts Practiced

| Concept | What it does | Example |
|---|---|---|
| if | Runs code if condition is True | if age >= 18 |
| else | Runs if condition is False | else: print('Minor') |
| elif | Checks another condition | elif marks >= 75 |
| and | Both conditions must be True | age >= 18 and has_id |
| or | At least one must be True | is_weekend or is_holiday |
| not | Flips True to False | if not is_raining |
| Nested if | if inside another if | Loan approval check |
| Ternary | One line if/else | result = 'Pass' if marks >= 40 else 'Fail' |
